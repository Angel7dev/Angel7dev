5.- Spaced repetition algorithm [backend].

supermemo2 spaced repetition algorithm

  • The letter is retrieved from the database by its id
  • check if it is the first review and update the flashcard data by the data obtained by the algorithm stored in review.
  • If it is not the first time and we have previous data we use it to determine the next review.
  • if the card has already been evaluated today just add a +1 to repetitions
  • It is always updated to today the last_review if the card is reviewed
